Class Description

Main

Our Main Class is named Main.In this class, first of all ,we show which features we have.User can choice any feature he want by clicking related number for this feature.Then by calling related method this job will be done.our Main class have 5 method,these are arithmeticCalculation() , others(), numberTransformation() , bitwiseOperation() , trigonometricFuntion().

ArithmeticCalculation

This class is contain 5 method summation(),substration(), multiplication(),division() and mod().summation() method can contain a series of numbers and displays their summtion. substraction() method also can contain a series of numbers and displays their substraction.Multiplication method also can contain a series of numbers and displays their multiplication.Division() and mod() Method can display division and modulas of two numbers if possible.

Others

This Class factorial(),square(),cube(),xToThePower(), squareRoot(),logarithm(),percentage() methods and displays their related result respectively.

NumberTransformation

This Class contains DecToBin(), DecToOct(), DecToHexa(), BinToDec(), BinToOct(), BinToHexa(), OctToDec(), OctToBin(), OctToHexa(), HexaToDec(), HexaToBin(), HexaToOct(). These Methods Transfer a number from one number system to another.

BitwiseOperation

This Class Contains 3 mathods such as OR(),AND() and X_OR() methods. These methods can compute bitwise OR,AND,X-OR operation of two number of Decimal,Binary,Octal and Hexadecimal Number System.Then displays this answer.

TrigonometricOperation

This Class Contains 3 methods such as Sin(),Cos(),Tan() methods and displays their related result.
